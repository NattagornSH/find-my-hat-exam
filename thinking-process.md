## ภาพรวมของเกม

เกมนี้คือ **เกมเดินบนตาราง** ผู้เล่นจะอยู่บนแผนที่สี่เหลี่ยมที่เต็มไปด้วยหลุม
เป้าหมายคือ **เดินไปเจอหมวก** โดยไม่ตกหลุมและไม่เดินออกนอกสนาม

---

## ขั้นตอนการทำงานทั้งหมด

---

### ขั้นที่ 1 — เตรียมของก่อนเริ่มเกม

```
กำหนดสัญลักษณ์ที่ใช้แทนสิ่งของต่างๆ บนแผนที่:
  ⭐️  = หมวก (เป้าหมายที่ต้องหา)
  🔥  = หลุม (อันตราย ห้ามเหยียบ)
  🍀  = พื้นว่าง (เดินได้ปลอดภัย)
  🐼  = ตัวผู้เล่น (ตำแหน่งที่เรายืนอยู่ตอนนี้)
```

---

### ขั้นที่ 2 — สร้างแผนที่แบบสุ่ม (generateField)

```
รับค่ามา 3 อย่าง:
  - ความสูงของแผนที่ (จำนวนแถว)
  - ความกว้างของแผนที่ (จำนวนคอลัมน์)
  - เปอร์เซ็นต์ที่จะให้เป็นหลุม (ค่าเริ่มต้น = 20%)

เริ่มสร้างแผนที่ทีละแถว ทีละช่อง:
  สำหรับทุกช่อง → สุ่มตัวเลข 0-1
    ถ้าตัวเลขน้อยกว่าเปอร์เซ็นต์หลุม → ช่องนี้คือหลุม 🔥
    ถ้าไม่ใช่ → ช่องนี้คือพื้นว่าง 🍀

วางตัวผู้เล่น 🐼:
  สุ่มตำแหน่งในแผนที่
  ถ้าสุ่มได้หลุม → สุ่มใหม่
  ถ้าสุ่มได้พื้นว่าง → วางผู้เล่นลงตรงนั้น

วางหมวก ⭐️:
  สุ่มตำแหน่งในแผนที่
  ถ้าสุ่มได้ตำแหน่งเดียวกับผู้เล่น หรือเป็นหลุม → สุ่มใหม่
  ถ้าได้ช่องที่โอเค → วางหมวกลงตรงนั้น

ส่งแผนที่ที่สร้างเสร็จออกไปใช้งาน
```

---

### ขั้นที่ 3 — จำตำแหน่งผู้เล่น (constructor)

```
รับแผนที่ที่สร้างมา แล้ว:
  จำจำนวนแถวและคอลัมน์ทั้งหมดของแผนที่

  ไล่ดูทุกช่องในแผนที่ตั้งแต่ต้นจนจบ:
    ถ้าเจอช่องที่เป็นตัวผู้เล่น 🐼 → จำตำแหน่งแถวและคอลัมน์ของช่องนั้นเอาไว้
```

---

### ขั้นที่ 4 — แสดงแผนที่บนหน้าจอ (print)

```
พิมพ์เส้นขีดบน

วนแสดงผลทีละแถว:
  นำทุกช่องในแถวมาต่อกัน แล้วพิมพ์ออกมาเป็นบรรทัด

พิมพ์เส้นขีดล่าง
```

---

### ขั้นที่ 5 — รับคำสั่งจากผู้เล่นและเดิน (moveUp / moveDown / moveLeft / moveRight)

```
ผู้เล่นกดปุ่ม:
  W → เดินขึ้น   = ลดค่าแถวลง 1 (ขยับไปบรรทัดบนสุด)
  S → เดินลง    = เพิ่มค่าแถวขึ้น 1 (ขยับลงมาบรรทัดล่าง)
  A → เดินซ้าย  = ลดค่าคอลัมน์ลง 1 (ขยับไปทางซ้าย)
  D → เดินขวา   = เพิ่มค่าคอลัมน์ขึ้น 1 (ขยับไปทางขวา)

หลังเดิน → เรียกตรวจสอบสถานะทันที (updateMap)
```

---

### ขั้นที่ 6 — ตรวจสอบผลหลังเดิน (updateMap)

```
เช็คอันดับแรก: ผู้เล่นเดินออกนอกแผนที่หรือเปล่า?
  ถ้าใช่ → แจ้ง "ออกนอกสนาม" แล้วจบเกม

ดูว่าช่องที่เพิ่งเดินมายืนคืออะไร:

  ถ้าเป็นหลุม 🔥:
    แสดงแผนที่ให้เห็นว่าเดินไปตกตรงไหน
    แจ้ง "ตกหลุม" แล้วจบเกม (แพ้)

  ถ้าเป็นหมวก ⭐️:
    วางผู้เล่นทับตรงนั้น (เพื่อแสดงว่าไปถึงแล้ว)
    แสดงแผนที่สุดท้าย
    แจ้ง "เจอหมวกแล้ว" แล้วจบเกม (ชนะ)

  ถ้าเป็นพื้นว่าง 🍀:
    เปลี่ยนช่องนั้นเป็นตัวผู้เล่น 🐼 (ย้ายตำแหน่งบนแผนที่)
    เล่นต่อได้ปกติ
```

---

### ขั้นที่ 7 — ลูปหลักของเกม (playGame)

```
แสดงข้อความต้อนรับและวิธีเล่น

สร้างแผนที่สุ่มขนาด 8x8 พร้อมหลุม 20%

วนซ้ำไปเรื่อยๆ จนกว่าเกมจะจบ:
  1. แสดงแผนที่ปัจจุบันให้ผู้เล่นดู
  2. ถามผู้เล่นว่าจะเดินทางไหน (w / a / s / d)
  3. เช็คว่ากดปุ่มอะไร แล้วเรียกฟังก์ชันเดินที่ตรงกัน
  4. ถ้ากดปุ่มอื่นที่ไม่ใช่ 4 ตัวนี้ → แจ้งว่าปุ่มผิด แล้วถามใหม่
  (วนกลับไปข้อ 1 ใหม่)

เกมจะหยุดเองเมื่อผู้เล่นแพ้หรือชนะ (จากฟังก์ชัน updateMap)
```

## สรุปภาพรวมแบบสั้นๆ

```
เริ่มเกม
  → สร้างแผนที่สุ่ม (วางหลุม วางผู้เล่น วางหมวก)
  → แสดงแผนที่
  → รอรับปุ่มจากผู้เล่น
  → เดิน
  → เช็คผล
      → ออกนอกสนาม? → แพ้ → จบ
      → เหยียบหลุม?  → แพ้ → จบ
      → เจอหมวก?    → ชนะ → จบ
      → พื้นว่าง?    → เดินต่อ → กลับไปแสดงแผนที่
```
